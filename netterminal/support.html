Common questions

Why does the LAN scan show no devices? Make sure you've granted local network permission in Settings → NetTerminal → Local Network. Some IoT devices only respond to ICMP and won't appear; this is an iOS platform limitation.

Why does dig sometimes return no records? The query is racing AliDNS, DNSPod, and Cloudflare; if all three time out, no result is shown. Ensure you have an active internet connection.

Pro upgrade purchase doesn't work. Make sure you're signed into the App Store with a valid Apple ID. If issues persist, email us.

Restore purchases. Go to Pro upgrade page, tap "Restore Purchases" at the bottom.
